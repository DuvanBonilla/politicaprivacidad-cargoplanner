Política de Privacidad para la aplicación CargoPlanner

Esta política de privacidad se aplica a la aplicación móvil CargoPlanner.Es una aplicación desarrollada para optimizar la gestión de operaciones portuarias de forma eficiente y segura. Respetamos la privacidad de los usuarios y garantizamos el uso responsable de la información conforme a las políticas de Google Play.

1. Uso de la información

La aplicación recopila únicamente los datos necesarios para el funcionamiento de las operaciones portuarias. Esto puede incluir información como nombre, cargo, ubicación de trabajo y registros de tareas operativas. No se recopilan ni almacenan datos personales sensibles, ni se comparte esta información con terceros no autorizados.

2. Permisos requeridos

  • Conexión a Internet: Necesaria para la sincronización de datos en tiempo real.
  • Almacenamiento (lectura y escritura): Se requiere acceso al almacenamiento del dispositivo para generar, guardar y compartir reportes de tareas operativas en formato Excel, creados por el usuario dentro de la aplicación. Estos archivos se almacenan localmente y pueden ser enviados por el medio que el usuario elija.

3. Modo Offline

La app permite trabajar sin conexión a internet. Los datos y reportes se almacenan temporalmente en el dispositivo y se sincronizan o comparten cuando haya conexión disponible.

4. Compartición de datos

Los reportes generados y cualquier otro dato ingresado por el usuario solo se comparten mediante canales que el propio usuario seleccione (por ejemplo, correo electrónico, mensajería o servicios en la nube). CargoPlanner no envía datos a servidores externos ni comparte información con terceros.

5. Cambios en la política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será publicado en esta misma página.
